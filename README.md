Shadowsocks软件介绍
SS，全称为 Shadowsocks（下文简称为SS），是一种基于 Socks5 代理方式的网络数据加密传输包的工具，可以使用其于科学上网 也就是翻墙 ，是目前最稳定，最常用的科学上网工具之一，全平台支持

SSR，全称为 ShadowsocksR（下文简称为SSR） 来历：

ShadowsocksR是breakwa11发起的Shadowsocks分支，在Shadowsocks的基础上增加了一些数据混淆方式，称修复了部分安全问题并可以提高QoS优先级。后来贡献者Librehat也为Shadowsocks补上了一些此类特性，甚至增加了类似Tor的可插拔传输层功能。

ShadowsocksR开始时曾有过违反GPL、发放二进制时不发放源码的争议，不过后来ShadowsocksR项目由breakwa11转为了与Shadowsocks相同的GPL、Apache许可证、MIT许可证等多重自由软件许可协议。

2017年7月19日，ShadowsocksR作者breakwa11在Telegram频道ShadowsocksR news里转发了深圳市启用SS协议检测并被大量用户转发，引发恐慌。7月24日，breakwa11发布了闭源的SS被动检测程序，引发争议。 2017年7月27日，breakwa11遭到自称 “ESU.TV” 的不明身份人士人身攻击，对方宣称如果不停止开发并阻止用户讨论此事件将发布更多包含个人隐私的资料，随后breakwa11表示遭到对方人肉搜索并公开个人资料的是无关人士，为了防止对方继续伤害无关人士，breakwa11将删除GitHub上的所有代码、解散相关交流群组，停止ShadowsocksR项目。但项目已被多人fork。并有人在其基础上继续发布新的版本，例如SSRR

Shadowrocket使用指南
1. 如何使用Shadowrocket
下载与安装：在App Store中搜索并下载Shadowrocket。

添加代理：

打开应用，点击右上角的“+”按钮。
输入代理服务器的地址、端口、密码和加密方式等信息，完成后点击“保存”。
启用代理：

选择刚添加的代理配置。
点击开关按钮，启用代理。
测试连接：在浏览器中访问被屏蔽的网站，确认代理是否正常工作。

2. Shadowrocket服务订阅
服务商选择：通过网上的服务商购买一个代理服务订阅。常见的服务商包括V2Ray、Shadowsocks等。
获取订阅链接：购买后，服务商通常会提供一个订阅链接，用于快速导入配置。
3. Shadowrocket路由器
Shadowrocket本身是应用程序，不能直接在路由器上使用。但你可以在路由器上设置Shadowsocks或V2Ray等协议，达到类似的效果：

路由器配置：选择支持VPN或代理的路由器，进入路由器管理界面，添加代理设置。
使用VPN功能：一些路由器支持VPN客户端，可以通过VPN服务实现科学上网。
4. Shadowrocket服务器订阅
添加订阅：
在Shadowrocket主界面，点击“配置”。
选择“添加订阅”，输入你的订阅链接，然后点击“保存”。
更新配置：添加后，应用会自动获取最新的服务器列表。
5. Shadowrocket如何订阅
使用订阅链接：
从你的代理服务商获取订阅链接。
在Shadowrocket中，选择“添加订阅”，输入链接，保存后会自动导入配置。
手动添加：如果没有订阅链接，也可以手动输入服务器信息。
6. Shadowrocket上不了
如果你发现Shadowrocket无法使用，可能有以下原因：

服务器问题：所使用的服务器可能已失效或被封锁，尝试更换其他服务器。
网络设置：检查设备的网络连接，确保连接正常。
配置错误：确认代理设置中的信息是否准确，包括地址、端口和加密方式等。
软件更新：确保应用已更新到最新版本，必要时尝试重新安装。
